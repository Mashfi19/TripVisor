# 🌍 TripVisor

TripVisor is a desktop-based **Travel Agency Management System** developed with **C# Windows Forms** and **SQL Server**. It was built as an academic project to simplify travel planning and agency management through an easy-to-use interface.

The system allows customers to explore travel packages, book hotels, make payments, and manage their bookings, while administrators can manage destinations, packages, hotels, customers, and booking information from a centralized dashboard.

---

## ✨ Features

* Secure user login and registration
* Browse travel packages and destinations
* Hotel booking system
* Package booking management
* Payment management
* Customer booking history
* Destination and hotel management
* Admin dashboard
* Manager and employee roles
* SQL Server database integration

---

## 🛠️ Technologies Used

* C#
* .NET Framework (Windows Forms)
* SQL Server
* MetroModernUI

## 🚀 Getting Started

1. Clone this repository.

```bash
git clone https://github.com/Mashfi19/TripVisor.git
```

2. Open the solution (`.sln`) file in **Visual Studio**.

3. Restore the required NuGet packages.

4. Update the SQL Server connection string if needed.

5. Run the project.

---

## 👥 User Roles

* **Owner** – Full control of the system.
* **Manager** – Manages packages, hotels, bookings, and employees.
* **Employee** – Assists with bookings and customer support.
* **Customer** – Explores packages, books trips, and makes payments.
